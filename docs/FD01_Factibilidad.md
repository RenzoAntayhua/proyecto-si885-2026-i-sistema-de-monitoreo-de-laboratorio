# FD01 - Informe de Factibilidad

## 1. Resumen Ejecutivo
El presente informe analiza la viabilidad técnica, operativa y económica para la implementación del sistema de telemetría Sysmon.

## 2. Factibilidad Técnica
El proyecto utiliza tecnologías Open Source (Wazuh, PostgreSQL, Terraform, Python) y servicios de nube estándar (Azure), asegurando alta disponibilidad.

## 3. Factibilidad Económica (Análisis Financiero)
Utilizando la calculadora de Microsoft Azure para una VM `Standard_B2s` (2 vCPUs, 4 GiB RAM), almacenamiento SSD de 30GB y ancho de banda estimado, el costo mensual proyectado es de aproximadamente **$35 USD**. Este costo es cubierto inicialmente por los créditos educativos de Azure for Students.
El uso de software Open Source evita costos de licenciamiento (Wazuh, PostgreSQL, React).

## 4. Conclusión
El proyecto es altamente factible.
